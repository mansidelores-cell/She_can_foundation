# She Can Foundation - Web Development Task

A clean, modern, and highly responsive single-page website built for the **She Can Foundation** internship application. This project is designed to be visually appealing, user-friendly, and lightweight.

---

## 🛠️ Features Included

### Core Requirements (Simple & Neat)
* **Home Section:** A welcome hero banner with a clear value proposition and a professional call-to-action button.
* **About Section:** A structured mission and vision grid overviewing the NGO's core purpose.
* **Visual Asset:** Integration of a high-quality, open-source banner image representing women empowerment.
* **Volunteer / Join Us Button:** Easily accessible conversion points throughout the layout.
* **Footer:** A clean footer layout containing copyright information and social media links.

### Bonus / Advanced Features Included
* **Responsive Design:** Fully optimized layout structure using CSS Media Queries to support seamless viewing on mobile phones, tablets, and desktop displays.
* **Interactive Mobile Menu:** A custom JavaScript hamburger toggle menu for smaller screens.
* **Form Validation:** Client-side JavaScript validation on the Volunteer application form to provide real-time status feedback without refreshing the page.
* **Clean UI Layout:** Designed using an elegant, modern color scheme (Empowerment Purple) and polished custom micro-interactions.

---

## 💻 Tech Stack Used

* **HTML5** – Semantic structure and layout.
* **CSS3** – Custom styling, CSS Variables, Flexbox, Grid systems, and Media Queries for complete responsiveness.
* **JavaScript (ES6)** – Interactive DOM manipulation, form validation, and mobile navigation toggle mechanics.
* **FontAwesome** – Scalable UI iconography.
